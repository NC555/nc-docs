# NC Docs

> A comprehensive and developer-friendly documentation website template built with Docusaurus.

A modern, responsive documentation hub designed to cover a wide range of technical topics from infrastructure and cloud to cybersecurity and development best practices.

## ✨ Features

- 📚 **Comprehensive Content Structure**: Organized sections for various technical domains.
- 📱 **Fully Responsive**: Optimized for a seamless experience on desktop, tablet, and mobile devices.
- 🔍 **Fast Full-Text Search**: Client-side search powered by `@easyops-cn/docusaurus-search-local`.
- 🌓 **Dark/Light Mode**: Built-in, user-toggleable theme support.
- 📝 **MDX Powered**: Write JSX components directly within your Markdown files.
- 📊 **Diagrams as Code**: Integrated Mermaid.js support for rendering diagrams from text.
- 🚀 **Optimized for Performance**: Built with Docusaurus for fast load times and a great user experience.
- 🔍 **SEO-Friendly**: Versioning, i18n, and clear content structure to improve search engine visibility.

## 📚 Documentation Topics

This template is populated with extensive documentation covering:

- **Ubuntu Private Lab**: Guides on Code Server, Coolify, Docker, HAProxy, NGINX, and system hardening.
- **Landing Zone Concepts**: Enterprise cloud architecture, Vault secret management, and discovery frameworks.
- **Microsoft Windows**: Ansible, Vault, Terraform, PowerShell, and WSL setup and management.
- **Kubernetes**: Architecture diagrams, deployment guides, and home lab setups.
- **Ansible**: Core concepts, playbooks for Ubuntu hardening, and network discovery.
- **Git**: Version control best practices, repository management, and branching strategies.
- **AWS**: In-depth coverage of fundamentals, architecture patterns, DevSecOps, and cost optimization.
- **Knowledge Base**: Essential concepts like API design, microservices, databases, and security patterns.
- **Cyber Security**: Frameworks for critical infrastructure, cloud security, IAM, incident response, and compliance.

## 🚀 Getting Started

### Prerequisites

- Node.js (v18.0 or later)
- npm, yarn, or pnpm package manager

### Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/nc-docs.git
    cd nc-docs
    ```
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm start
    ```
4.  Open [http://localhost:3000](http://localhost:3000) in your browser to see the result.
